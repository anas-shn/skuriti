# skuriti

Repositori ini berisi konfigurasi dan panduan instalasi untuk Snort dan Suricata pada sistem Linux (contoh: Ubuntu). Tujuan: menyediakan konfigurasi dasar, pengelolaan rules, dan langkah cepat untuk menjalankan kedua IDS/IPS tersebut.

## Struktur repositori

- [README.md](README.md) — ringkasan proyek dan panduan cepat
- [README.Snort.md](README.Snort.md) — panduan instalasi & konfigurasi Snort
- [README.Suricata.md](README.Suricata.md) — panduan instalasi & konfigurasi Suricata
